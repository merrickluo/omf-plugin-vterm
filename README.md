<br/>


## Install

```fish
$ omf install vterm
```


## Usage

in vterm inside emacs, assume you won't need to open another emacs inside emacs

```fish
$ find_file file # = (find-file file)
$ emacs file # alias of find_file
$ e file # alias of find_file

$ say "hello" # = (message "hello")
```
